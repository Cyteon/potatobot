# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 3.x.x   | :white_check_mark: |
| 2.x.x   | :white_check_mark: |
| < 2.0   | :x:                |

## Reporting a Vulnerability

Report a vulnerability by making a ticket in https://discord.gg/wtur9j8uVP.
Wait for an owner before explaining the vulnerability
